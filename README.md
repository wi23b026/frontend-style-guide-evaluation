# Evaluating the Impact of Front-End Style Guide Usage on Accessibility and Usability in Web Applications
Development and evaluation of a frontend style guide, focusing on usability and accessibility.

## Reasearch Objective
- RQ1: Which challenges occur when working without a front-end style guide in terms of accessibility and usability?
- RQ2: How can accessibility and usability be addressed within a front-end style guide?
- RQ3: How does a web application based on a front-end style guide compare to one based directly on a corporate design manual in terms of accessibility and usability?

## Start Project
From each app directory...
- `\websites\website-with-style-guide`
- `\websites\website-without-style-guide`
- `\frontend-style-guide`

Install Angular CLI:
```bash
npm install -g @angular/cli
npm install
```

Start application:
```bash
ng serve
```
