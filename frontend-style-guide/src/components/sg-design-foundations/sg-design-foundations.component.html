Design Foundations

Layout Structure

Grid System

  • Desktop: 12-column grid
  • Tablet: 8-column grid
  • Mobile: 4-column grid

Container

max-width: 120rem;
margin: 0 auto;
padding: 0 1rem;

Layout Principles

  • Grid-aligned layout structure
  • Consistent spacing and component placement
  • Whitespace used intentionally
  • Minimal and structured layout design
  • Use the same visual rhythm for equivalent pages, such as search, form, summary and table order
  • Do not introduce page-specific visual systems unless the style guide is updated first

Standard Page Structure

  • Global navigation is placed at the top of every page
  • Page content is placed in a centred container with max-width: 120rem
  • Page title and primary controls appear before data-heavy sections
  • Forms appear before the related table when the form creates or edits table content
  • Search and filters appear directly above the table they affect
  • Feedback containers such as notifications and dialogs are global overlays, not nested inside cards
  • Primary page actions align with the page title or related section header
  • Secondary actions stay close to the content they affect
