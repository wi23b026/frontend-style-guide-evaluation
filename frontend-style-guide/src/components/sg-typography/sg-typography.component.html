Font Stack & Rules

  • --font-heading: "Heavitas", sans-serif
  • --font-body: Arial, Helvetica, sans-serif
  • Headings: Heavitas only
  • Body text: Arial only
  • No additional fonts allowed
  • Clear hierarchy (H1 > H2 > H3 > Body)
  • Maximum heading level: H3
  • Line height: 1.4-1.6 for readability
  • Body text must stay at 1rem minimum for primary content
  • Small text is reserved for labels, helper text and secondary information only
  • Do not scale type with viewport width; use the documented type scale at every breakpoint
  • Paragraphs and form help text should use concise wording and readable line lengths

Type Scale (8pt system)

  • --font-size-h1: 3rem
  • --font-size-h2: 2rem
  • --font-size-h3: 1.5rem
  • --font-size-body: 1rem
  • --font-size-small: 0.875rem
