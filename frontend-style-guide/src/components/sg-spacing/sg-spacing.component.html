Spacing System (8pt Grid)

--space-1: 0.5rem
--space-2: 1rem
--space-3: 1.5rem
--space-4: 2rem
--space-5: 3rem

Spacing Rules

  • Only multiples of 8 are allowed
  • No arbitrary spacing values
  • Consistent spacing across all components
  • Use spacing tokens instead of one-off margin or padding values
  • Related controls use --space-2 between items by default
  • Large page sections use --space-4 between sections by default
  • Components placed inside forms or toolbars must align to the same grid baseline
